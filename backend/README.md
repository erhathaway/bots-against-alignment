# Install

`poetry shell`

`poetry install`

# Run

`uvicorn src:app --reload`

# API Docs

After running on localhost:8000, go to:

Swagger style:

[http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)

or

Redoc style:

`[http://127.0.0.1:8000/redoc](http://127.0.0.1:8000/redoc)


# Tests

`pytest -vv -s`
