# Legacy frontend (deprecated)

This folder is kept for reference only. The active app is in `app/` (SvelteKit v2 + Svelte v5).

## Install Dependencies

`bun install`

# Build 

`bun run build`

# Run Dev

`bun run dev -- --open`

# Environment

Create `frontend/.env`:

```
VITE_BACKEND_API=http://127.0.0.1:8000
VITE_ENABLE_VERCEL_ANALYTICS=false
```
